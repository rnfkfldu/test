
# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers. The application will display 6 unique numbers between 1 and 45.

## Project Outline

*   **`index.html`**: The main HTML file containing the structure of the web page.
*   **`style.css`**: The CSS file for styling the application.
*   **`main.js`**: The JavaScript file containing the logic for generating lottery numbers.

## Current Plan

1.  **Create the basic HTML structure:** Add a title, a container for the numbers, and a button to generate new numbers.
2.  **Style the application:** Design a visually appealing layout with colors, fonts, and spacing.
3.  **Implement the number generation logic:** Write a JavaScript function to generate 6 unique random numbers and display them on the page.
